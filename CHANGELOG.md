# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-03-07

### Added
- **Share Room Feature**: New "Share" button in the lobby to copy join links.
- **Auto-Join Support**: Players can now join rooms directly via URL parameters (e.g., `?room=ABCD`).
- **Back to Menu Button**: Added to lobby, in-game header, and game-over screens for easier navigation.
- **Enhanced Chat Notifications**: Visual emojis and better system messaging when players join or leave.
- **Dice Roll Animation**: Added a "shake" animation to the dice during the rolling phase.

### Changed
- Improved game header layout with a consolidated menu button.
- Updated versioning across the project.

## [1.1.0] - 2026-03-06
- Initial feature set includes structured logging, fun bot names, and visual feedback improvements.
