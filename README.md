# 🎲 Ludo Multiplayer - Classic Board Game

## Features

### 🎮 Game Modes
- **2-4 Players**: Play with friends online
- **Classic Ludo Rules**: Traditional gameplay
- **Real-time Multiplayer**: WebSocket-based

### 🎯 Game Features
- ✅ Dice rolling with animation
- ✅ Token movement
- ✅ Capture opponent tokens
- ✅ Safe zones
- ✅ Home stretch
- ✅ Winner celebration
- ✅ Turn-based gameplay
- ✅ Auto-skip if no valid moves

### 🎨 Visual Features
- Beautiful board design
- Smooth animations
- Dice roll effects
- Token movement animations
- Mobile responsive
- PWA ready

### 🏆 Game Rules
1. Roll 6 to bring token out
2. Move tokens based on dice
3. Capture opponents by landing on them
4. Safe zones protect your tokens
5. First to get all 4 tokens home wins!

## How to Run

```bash
npm install
npm start
```

Visit: http://localhost:3000

## Tech Stack
- **Backend**: Node.js + Express + WebSocket
- **Frontend**: Vanilla JavaScript + Canvas/CSS
- **Game Logic**: Classic Ludo rules
